# Retail Management Database System

A SQL-based Retail Management System project designed to simulate real-world retail operations and business analytics.

## Features
- Retailers Management
- Products Management
- Customer Orders
- Sales Analysis
- Employee Records
- Suppliers & Payments

## SQL Concepts Used
- Joins
- Aggregate Functions
- Views
- Stored Procedures
- Indexing
- Primary & Foreign Keys

## Tools
- MySQL
- SQL
